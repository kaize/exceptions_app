# ExceptionsApp

ExceptionsApp is gem, that provides a simple way of handling exceptions in Rails applications.

## Installation

Add this line to your application's Gemfile:

    gem 'exceptions_app'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install exceptions_app
